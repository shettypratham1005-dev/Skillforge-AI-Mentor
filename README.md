# 🚀 SkillForge AI Mentor

An AI-powered Workplace Readiness Platform designed to help freshers become industry-ready by providing personalized learning roadmaps, resume analysis, workplace task guidance, interview preparation, and AI-powered mentoring.

---

## 📌 Project Overview

Freshers often face challenges in understanding workplace expectations, completing manager-assigned tasks, preparing for interviews, and building professional communication skills.

SkillForge AI Mentor acts as a virtual senior mentor that guides users through:

- Resume Analysis
- Personalized Learning Roadmaps
- Workplace Task Execution
- Industry Simulations
- AI Mentor Assistance
- Interview Preparation
- Professional Communication
- Portfolio Building

The platform helps freshers gain confidence and perform effectively during the early stages of their careers.

---

## 🎯 Problem Statement

Fresh graduates often possess theoretical knowledge but struggle with:

- Understanding real-world industry tasks
- Professional workplace communication
- Project execution workflows
- Interview preparation
- Career planning

SkillForge AI Mentor bridges this gap through AI-powered mentorship and workplace guidance.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/shettypratham1005-dev/skillforge-ai-mentor.git
```

### Move to Project Directory

```bash
cd skillforge-ai-mentor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

## ▶️ Run Application

```bash
streamlit run app.py
```

